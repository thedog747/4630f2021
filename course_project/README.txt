This folder contains all files related to the course project.

Under the folder "Proposal" you will find images and text relating to the planning and layout of my program.

The "source" folder contains the source code of my project which can be compiled in android studio.

The file "report.txt" contains the requested short report about the problem the project is attempting to solve, its goals, alternate approaches to solving the problem, the chosen approach, and a justification for choosing it.

The "build" folder has the requested .apk file

"build_instructions.txt" has instructions on how to build and run the app.

"operation_instructions.txt" has instructions on how to use the software.
